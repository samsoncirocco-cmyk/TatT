  <name>Task 1: Create Vertex AI embedding service and normalization utilities</name>
  <files>src/services/vertex-embedding-service.ts, src/lib/embedding-normalization.ts</files>
  <action>
Create `src/lib/embedding-normalization.ts`:
- Export `normalizeVector(vec: number[]): number[]` — L2 normalization to unit vector. Handle zero-magnitude edge case (return zero vector, don't divide by zero).
- Export `assertUnitVector(vec: number[], tolerance?: number): void` — throws if vector magnitude deviates from 1.0 by more than tolerance (default 0.01). Use for debugging/validation.

Create `src/services/vertex-embedding-service.ts`:
- Import `normalizeVector` from `../lib/embedding-normalization`.
- Use `google-auth-library` (`GoogleAuth`) for authentication — already available in the project via `@google-cloud/vertexai` dependency.
- Export `generateTextEmbedding(text: string): Promise<number[]>` — calls Vertex AI REST API endpoint `https://us-central1-aiplatform.googleapis.com/v1/projects/${PROJECT_ID}/locations/us-central1/publishers/google/models/text-embedding-004:predict`. Returns normalized 768-dimension embedding. PROJECT_ID from `process.env.GCP_PROJECT_ID`.
- Export `batchGenerateEmbeddings(texts: string[]): Promise<number[][]>` — batches up to 5 instances per request (Vertex AI limit). Returns array of normalized embeddings. Used by seed script.
- Export `getQueryEmbedding(queryText: string): Promise<number[]>` — wraps `generateTextEmbedding` with Firestore query cache. Cache key = lowercase trimmed query text. TTL = 7 days. Check `query_embedding_cache` Firestore collection first. On cache miss, generate via Vertex AI, store in cache with `FieldValue.serverTimestamp()`, return.
- Use `firebase-admin/firestore` for cache operations (import `getFirestore`, `FieldValue`).
- Add error handling: wrap fetch in try/catch, log errors with `[VertexAI]` prefix, throw meaningful errors.
- Do NOT use `@google-cloud/aiplatform` SDK — use raw fetch with GoogleAuth token (lighter weight per research).
  </action>

  </verify>
  <done>
