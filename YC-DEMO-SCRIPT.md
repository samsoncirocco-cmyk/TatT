# TatT — YC Demo Day Pitch Script
## Duration: 2 minutes | Target: YC W2026

### [0:00-0:15] THE HOOK
"A tattoo is permanent, but today the design process still feels like guesswork. TatT is the AI-native platform that lets people design, preview, and book with confidence before the needle ever touches skin."

### [0:15-0:30] THE PROBLEM
"This is a global market often estimated around $10B+, but the user experience is broken. People have commitment anxiety, iteration is slow, and regret rates are commonly reported near 40% in survey cohorts. There is still no default workflow for trying variations, seeing realistic placement, and instantly finding the right artist."

### [0:30-0:50] THE DEMO FLOW
Click: `Start in The Forge`  
Show: Prompt input + style/body placement controls  
Say: "I’m entering: 'fine-line phoenix forearm with smoke detailing.'"

Click: `Enhance with AI Council`  
Show: 3-agent enhancement output (Creative, Technical, Style)  
Say: "This is our differentiator: instead of one-shot prompting, TatT runs a multi-agent council that rewrites the prompt for composition, technical tattoo constraints, and style consistency."

Click: `Generate`  
Show: Multiple tattoo outputs ready for layer editing  
Say: "Now we generate production-ready variations from that council-enhanced prompt."

### [0:50-1:20] THE MAGIC MOMENT
Click: `Preview in AR`  
Show: Design overlaid on forearm with placement metadata  
Say: "Here’s the same design on-body in real time. This is where commitment anxiety drops, because users can actually see fit and scale before they commit."

Click: `Find Artists`  
Show: Instant ranked artist list with match reasoning  
Say: "And instantly, TatT matches this design to artists using hybrid ranking: semantic vectors in Supabase plus relationship intelligence in Neo4j. In one flow, we go from idea to visual confidence to artist action."

### [1:20-1:40] TRACTION
"We have the Phase 1 MVP in production at tatt-production.up.railway.app. The core pipeline is live: Council enhancement, generation, AR preview endpoint, stencil export, and hybrid artist matching. We’ve built on a 100-artist matching dataset with a production-ready migration path to Vertex text embeddings and a tested engineering base across nearly 200 automated tests."

### [1:40-2:00] THE ASK
"We’re raising a **$2.5M seed** to become the category leader in AI-powered tattoo planning and artist discovery. The timing is now: generative quality is finally good enough, consumer comfort with AI creation is mainstream, and tattoo is a high-intent vertical with clear transaction value. We’re building the default front door for modern tattoo decisions."
